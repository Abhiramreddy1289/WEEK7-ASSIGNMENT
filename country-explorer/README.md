# Country Explorer

A React application that allows users to explore countries from around the world using data from the REST Countries API.

## Features
- **Fetch Countries**: Automatically fetches a comprehensive list of countries on load.
- **Search**: Filter countries by name in real-time.
- **Detailed Cards**: View country flags, capitals, populations, and regions.
- **Responsive Design**: Styled with Tailwind CSS for a modern, responsive interface.

## Technologies Used
- React
- Vite
- Tailwind CSS
- REST Countries API

## Getting Started
1. Navigate to the `country-explorer` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
