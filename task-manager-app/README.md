# Task Manager App

A simple and intuitive task management application built with React, helping users organize their daily tasks efficiently.

## Features
- **Task Management**: Add and track tasks through a clean interface.
- **Modern UI**: Styled with Tailwind CSS for a professional look and feel.
- **Responsive**: Works across various devices and screen sizes.

## Technologies Used
- React
- Vite
- Tailwind CSS

## Getting Started
1. Navigate to the `task-manager-app` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
