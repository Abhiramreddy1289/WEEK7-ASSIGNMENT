# WEEK7 ASSIGNMENT

This repository contains multiple small React + Vite projects.

## Projects

### 1) Country Explorer
A React app to explore countries (search/browse UI).

- Location: `country-explorer/`
- Tech: React + Vite

### 2) Products Page
A products listing page with filtering/sorting and responsive layout.

- Location: `products-page/`

### 3) Task Manager App
A task manager application (create/list/update tasks).

- Location: `task-manager-app/`

## How to run a project

From the repository root, go into the project folder you want to run and start Vite:

```bash
cd <project-folder>
npm install
npm run dev
```

Then open the shown local URL in your browser.
