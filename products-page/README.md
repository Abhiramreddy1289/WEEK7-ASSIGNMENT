# Products Page

A React-based web application designed to display a collection of products in a clean and organized layout.

## Features
- **Product Listing**: Displays products using a modular component structure.
- **Responsive Layout**: Optimized for different screen sizes.
- **Fast Development**: Built on Vite for a quick and efficient development experience.

## Technologies Used
- React
- Vite
- CSS3

## Getting Started
1. Navigate to the `products-page` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
